PROJECT TITLE: Effects of COVID-19 on trade

PURPOSE OF PROJECT: Download a file with data from the web and process the data. The processed data are stored in files and in a mysql database and are presented to the user through a gui.

DATE: Jun 2023

AUTHORS: Spyros


USER INSTRUCTIONS:
Change the "mysqlconfig.txt" file so that the creation of the database succedes.

To execute the python file "app.py", type the following command in the command line. (Being in the same directory as the file "app.py")
-> python app.py


Note:
-Python version: 3.7.9
-The file "data.csv" is not necessary if you have internet connection while running the code as it is downloaded from the "app.py" script.
